## 基础语法

### 标题

使用 `#` 号来创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**：使用 `**文本**`
- *斜体*：使用 `*文本*`
- ~~删除线~~：使用 `~~文本~~`

### 列表

无序列表：
- 项目一
- 项目二
- 项目三

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

### 代码

行内代码使用反引号：`code`

代码块使用三个反引号：

```javascript
function hello() {
  console.log('Hello, World!')
}
```

## 进阶技巧

### 表格

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |

### 引用

> 这是一段引用文字。
> 可以跨越多行。

### 任务列表

- [x] 已完成的任务
- [ ] 待完成的任务

## 总结

Markdown 语法简单易学，非常适合用来写博客文章。掌握了这些基础语法，你就可以开始创作了！
